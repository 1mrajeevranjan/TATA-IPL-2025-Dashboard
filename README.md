## 📌 Project Overview

**TATA IPL 2025 Live Dashboard** is a premium, high-performance analytics platform designed to provide cricket enthusiasts and analysts with real-time insights into the 2025 Indian Premier League season. Built with a focus on visual excellence and data density, this dashboard offers a comprehensive suite of features ranging from live match tracking to deep-dive player performance analytics.

Designed for a seamless experience across all devices, the dashboard utilizes modern web technologies to deliver a "glassmorphic" dark-mode interface that feels alive and responsive.

---

## ✨ Key Features

- **🔴 Live Match Center**: Real-time score updates, dynamic win probability charts, and "on-strike" batsman tracking.
- **📊 Advanced Analytics**: Interactive seasonal scoring trends and run-rate distribution visualizations using ApexCharts.
- **🏆 Tournament Race**: Real-time leaderboards for the Orange Cap (top run-scorers) and Purple Cap (top wicket-takers).
- **📈 Comprehensive Standings**: A detailed 10-team points table with Net Run Rate (NRR) and qualification indicators.
- **👤 Player Profiles**: Dedicated sections for individual player statistics and performance history.
- **⚔️ Team Comparison**: Side-by-side analytical tools to compare franchise strengths and weaknesses.
- **🌗 Responsive Design**: Fully optimized for mobile, tablet, and desktop viewports with a collapsible premium sidebar.

---

## 🛠️ Technology Stack

- **Core**: Semantic HTML5 & Vanilla JavaScript (ES6+)
- **Styling**: Modular Vanilla CSS with a custom Design Token System
- **Data Visualization**: [ApexCharts](https://apexcharts.com/) for interactive data storytelling
- **Icons**: Google Material Symbols & Material Icons
- **Typography**: Space Grotesk (Headings), Manrope (Body), and Inter (Data)

## 🎨 Design System

The application follows a strict **Design Token Architecture**, ensuring consistency across all pages. 

- **Theme**: Dark Mode (default) with deep surface colors and vibrant primary accents.
- **Elevation**: 4-level shadow system (`shadow-sm` to `shadow-lg`).
- **Typography**: A clear hierarchy separating display fonts for KPIs and clean sans-serif fonts for tabular data.
- **Grid System**: Flexible CSS Grid shell with an 8-column/4-column split for information-rich views.
